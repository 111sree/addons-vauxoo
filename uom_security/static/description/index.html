UOM Security

Unit of Measures visibility by area

Odoo add a functionality to mange multiple unit of measures in the system, one apply this permission the unit of measure will be shown in the system. This module create three groups to manage unit of measure.

  • Purchase UOM
  • Sales UOM
  • Warehouse UOM

In this way we can control when to show the unit of measure in certain areas of the instance.
